5. CÓMO FUNCIONAN LOS BOTONES
================================================================================

FLUJO DE NAVEGACIÓN:

1. Usuario hace clic en botón → href="{{ route('tesoreria.view.pazysalvo') }}"
2. Blade genera URL → /tesoreria/paz-y-salvo
3. Navegador envía GET request a esa URL
4. Laravel router captura con Route::get('/paz-y-salvo', ...)
5. Router invoca TesoreroController@viewPazYSalvo()
6. Controlador ejecuta: return view('tesoreria.paz_y_salvo')
7. Blade renderiza la vista y devuelve HTML al navegador
8. Usuario ve la página con el contenido

VENTAJAS DE ESTE ENFOQUE:

✓ URLs dinámicas: Si cambias el nombre de ruta, todos los botones se actualizan
✓ Seguridad: Middleware auth protege todas las rutas
✓ Mantenimiento: Un solo lugar para cambiar rutas (routes/web.php)
✓ Escalabilidad: Fácil de agregar nuevas funciones
