Cómo Funcionan los Botones

En el archivo dashboard.blade.php, cada botón usa el helper Blade route() para generar URLs dinámicas:

<a href="{{ route('tesoreria.view.pazysalvo') }}" class="btn btn-success">
Esta línea genera: /tesoreria/paz-y-salvo

Próximos Pasos

  1. Acceder al Dashboard: http://localhost/tesoreria/dashboard
  2. Hacer clic en cada botón: Cada uno navegará a su vista correspondiente
  3. Verificar URLs en navegador: Las rutas deben ser correctas
  4. Validar en consola (F12): No debe haber errores de rutas
  5. Implementar lógica AJAX: Conectar las vistas con los métodos API

Nota: Las rutas están protegidas por middleware auth. Debes estar autenticado para acceder a ellas.
